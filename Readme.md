
# Pyrite!

-- A blog generator 


Testing can be performed by running 

        python -m unittest discover
        
or

        ./do_tests.sh

or by running the python module nose....

To create the pyrite graph, install Graphviz and then run:
        
        ./create_graphs.py

To cleanup vim files and pyc files, run
    
        ./cleanup
